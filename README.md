# Rust-learning
This repository is for learning Rust Programming Language

## Installation

```
curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
```

## Useful Commands

- Compile using
    ```
    rustc main.rs
    ```

### Cargo 

- To create a new Rust project:
    ```
    cargo new project_name
    ```

- To build the project:
    ```
    cargo build
    ```

- To run the project:
    ```
    cargo run
    ```

- To check for errors without building:
    ```
    cargo check
    ```
